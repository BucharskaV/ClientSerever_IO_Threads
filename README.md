# Chat Application (Client–Server, Java Sockets)
A multi-client chat application developed in Java using TCP sockets and a client–server architecture. The application supports real-time messaging, private messaging (to one or multiple users), and selective broadcasting.
## Technologies & Techniques
- Java
- TCP/IP Sockets
- Multithreading
- Synchronization
- Client–Server Architecture
- Event-Driven Programming
- Stream I/O
- Basic Security Filtering
Project includes a Swing-based graphical user interface for both client and server sides.
